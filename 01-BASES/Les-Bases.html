Les Balises DIV :

La Balise DIV, est une balise neutre de type block. Elle me servira pour structurer ma page.

Les Balises Block agissent sur un Axe Vertical et donc génère un retour à la ligne.
Je pourrais toutes-fois changer leurs comportements avec du CSS.

Les Balises Inline elles, agissent sur un Axe Horizontal et donc ne génère aucun retour à la ligne.
Je pourrais toutes-fois changer leurs comportements avec du CSS.

Les Formulaires :

  • La Balise FORM permet de créer le formulaire.
  • name : Permet de donner un nom au formulaire
  • action : Cet attribut est OBLIGATOIRE et permet de renseigner le script / la page vers laquelle l'utilisateur sera redirigé lors de l'envoi du Formulaire. Cette page va reçevoir l'ensemble des données saisie par mon utilisateur.
  • method : Permet de préciser quelle méthode [GET | POST] doit être utilisé par le navigateur pour transmettre les données saisies par l'utilisateur vers la page ACTION.














Créer un lien en HTML

  • La balise "a" permet de créer un lien vers une autre ressource. (Images, Vidéos, PDF, ZIP, ...

  • l'Attribut href : Permet de définir le chemin vers le fichier à lier.

  • l'Attribut target : Prend plusieurs valeurs : "_blank" pour ouvrir le lien dans une nouvelle fenêtre par exemple...

  • l'Attribut title : Permet de donner un titre au lien. Ce titre sera afficher dans une petite bulle au survol du line par la souris.
