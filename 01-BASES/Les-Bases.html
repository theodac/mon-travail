Les Balises Div :

La Balise DIV , est une balise neutre de type block . Elle me servira pour structurer ma page .

Les Balises Block agissent sur un Axe Vertical et donc génère un retour a la ligne .
Je pourrais toutes-fois changer leurs comportements avec du CSS

Les Balises Inline elles , agissent sur un Axe Horizontal et donc ne génère aucun retour a la ligne Je pourrais toute-fois changer leurs comportements avec du CSS , Les balises inline doivent toujours être entourés de balises de type block.

Les Formulaires :

  • La Balise FORM permet de créer le formulaire.
  • name : Permet de donner un nom au formulaire
  • action : Cet attribut est OBLIGATOIRE et permet de renseigner le script/ la page vers laquelle l'utilisateur sera redirigé lors de l'envoi du Formulaire. Cette page va recevoir l'ensemble des données saisie par mon utilisateur
  • method : Permet de préciser quelle méthode [Get | Post] doit être utilisé par le ,avigateur pour transmettre les données saisie par l'utilisateur vers la page Action.













Créer un lien en HTML

  • la balise "a" permet de cree un lien vers un autre ressource. (Image,Vidéo,Pdf,Zip...)

  • L'Attribut href: Permet de definir le chemin vers le fichier à lier.

  • L'Attribut target : Prend plusieurs valeurs : "_blank" pour ouvrir le lien dans une nouvelle fenêtre par exemple...

  • L'Attribut title : Permet de donner un titre au lien. Ce titre sera afficher dans une petite bulle au survol du line par la souris
